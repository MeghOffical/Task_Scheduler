# Project Description – *PlanIt : Task Scheduler App*

Our project is about building a **Task Scheduler app** called *PlanIt*. The main goal of this app is to help users like **students, working professionals, and freelancers** manage their daily tasks in a simple and distraction-free way. Users can create, update, delete, and complete tasks, and all tasks will be stored safely in a **local database**, so they are not lost after restarting the computer.

The app will provide useful features like **reminders/notifications**, **task filtering**, and **progress tracking** so users can stay organized and motivated. It will also include an optional **AI chatbot assistant**, which allows users to add or check tasks using natural language, for example typing *“Remind me to finish assignment tomorrow at 6 pm”*. This makes the app faster and more user-friendly.

Overall, *PlanIt* aims to be a **lightweight, easy-to-use, and reliable desktop task manager**. By focusing on both **functional needs** (CRUD, reminders, export, chatbot) and **non-functional needs** (speed, privacy, accessibility), the project ensures that users can manage their tasks effectively, without unnecessary complexity.
