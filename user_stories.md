<h2>High Priority</h2>

<font size="5">**User Story 1: Task Creation**</font>

**Front:**  
As a user, I want to create new tasks so that I can keep track of my responsibilities.

**Back:**  
- **Success:**  
  - Task appears in the main list.  
  - Task is saved immediately.  

- **Failure:**  
  - Task not saved  
  - Task does not appear or disappears after refresh.
    

<font size="5">**User Story 2: Task Editing**</font>

**Front:**  
As a user, I want to edit existing tasks so that I can update them when plans change.

**Back:**  

- **Success:**  
  - User edits title/description/deadline/priority.  
  - Updated task is displayed instantly.  

- **Failure:**  
  - Changes not saved.  
  - Old values still show after editing.
    

<font size="5">**User Story 3: Task Deletion**</font>

**Front:**  
As a user, I want to delete tasks so that I can remove irrelevant or completed work.

**Back:**  

- **Success:**  
  - Task is removed from the list permanently.  
  - Confirmation message shown.  

- **Failure:**  
  - Task still appears after deletion.  
  - Wrong task gets deleted.  
 

<font size="5">**User Story 4: Priority Setting**</font>

**Front:**  
As a user, I want to assign priorities (High, Medium, Low) so that I can focus on important tasks first.

**Back:**  

- **Success:**  
  - User can assign/change priority.  
  - Tasks displayed in correct priority order.  

- **Failure:**  
  - Priority not saved.  
  - Sorting shows wrong order.  

<font size="5">**User Story 5: Deadline Management**</font>

**Front:**  
As a user, I want to set deadlines so that I don’t miss important dates.

**Back:**  

- **Success:**  
  - User can set deadline.  
  - Reminder/notification appears before deadline.

- **Failure:**  
  - No reminder delivered.  
  - Wrong deadline shown.

<font size="5">**User Story 6: Sorting & Viewing**</font>

**Front:**  
As a user, I want to sort and filter tasks by priority or deadline so that I can organize my work effectively.

**Back:**  

- **Success:**  
  - Sorting works correctly (priority or deadline).  
  - After filtering, only the tasks which match the filter are shown.

- **Failure:**  
  - Sorting does not change order.  
  - Filter hides or shows wrong tasks.

<font size="5">**User Story 7: Reliability & Security**</font>

**Front:**  
As a user, I want my tasks to be stored securely so that I don’t lose important information.

**Back:**  

- **Success:**  
  - Data is saved persistently (after app restart).  
  - No data loss.

- **Failure:**  
  - Tasks disappear after restart.  
  - Data loss.

<h2>Medium Priority</h2>

<font size="5">**User Story 8: Search Tasks**</font>

**Front:**  
As a user, I want to search for tasks by title or keyword so that I can quickly find what I need.

**Back:**  

- **Success:**  
  - Search returns correct matching tasks.  
  - No delay in showing results.

- **Failure:**  
  - Search gives no results even when matches exist.  
  - Wrong tasks appear in results.
 
<font size="5">**User Story 9: Notifications**</font>


**Front:**  
As a user, I want to receive notifications/reminders before deadlines so that I don’t miss tasks.

**Back:**  

- **Success:**  
  - Notification is triggered at set time (e.g., 10 min before deadline).  
  - Notifications work even if app is minimized.

- **Failure:**  
  - No notification received.  
  - Wrong notification timing.

<font size="5">**User Story 10: Performance**</font>


**Front:**  
As a user, I want the app to respond instantly so that I can manage tasks quickly.

**Back:**  

- **Success:**  
  - Task creation/editing/deletion happens in under few seconds.  
  - Sorting is immediate.

- **Failure:**  
  - Lag or freezing when adding/sorting tasks.  
  - App crashes on normal use.
 
<font size="5">**User Story 11: Task Categories/Tags**</font>

**Front:**  
As a user, I want to categorize my tasks (e.g., Work, Study, Personal) so that I can organize them better.

**Back:**  

- **Success:**  
  - User can add/edit/delete categories.  
  - Tasks show under the correct category.

- **Failure:**  
  - Task appears in wrong category.  
  - Categories disappear after restart.

<h2>Low Priority</h2>

<font size="5">**User Story 12: Multi-Device Access (Future)**</font>

**Front:**  
As a user, I want to access my tasks on multiple devices so that I can stay updated anywhere.

**Back:**  

- **Success:**  
  - Tasks sync across devices automatically.  
  - Changes on one device reflect everywhere.

- **Failure:**  
  - Tasks are missing on other devices.  
  - Conflicting versions appear.
 
<font size="5">**User Story 13: Intelligent Suggestions (Future LLM Integration)**</font>

**Front:**  
As a user, I want the app to suggest deadlines or priorities using AI so that I can plan better.

**Back:**  

- **Success:**  
  - AI suggestions appear while creating a task.  
  - User can accept or reject suggestions.

- **Failure:**  
  - Suggestions are irrelevant.  
  - No option to override AI recommendations.
